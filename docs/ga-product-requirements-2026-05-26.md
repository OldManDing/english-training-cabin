# 英语训练舱 GA 产品需求与功能闭环

日期：2026-05-26

## 1. 产品定位

英语训练舱不是“展示型题库页”，而是本地优先的英语备考训练产品。GA 阶段的最低目标是：一个 CET-4 学习者不需要开发者说明，也能从首次进入开始完成诊断、练习、复习和进度查看，并知道下一步该练什么。

当前公开口径：

- 首发聚焦 CET-4 训练闭环。
- 内置内容为原创模拟材料和核心词表，不声称提供官方真题库。
- 支持用户导入自有或授权 JSON 材料。
- AI 用于反馈、生成模拟材料和口语/主观题辅助；AI 不可用时必须有可用兜底，不能中断训练。

## 2. 核心用户旅程

### 2.1 首次进入

用户首次进入后应看到“今日训练”工作台，而不是营销页。工作台必须说明：

- 当前目标考试和目标分。
- 是否已有能力证据。
- 如果没有能力证据，首要动作必须是入门诊断。
- 学习记录存储位置和内容来源边界。

首次闭环：

```text
进入今日训练
-> 开始入门诊断
-> 设置目标分、考试日期、每日时间
-> 完成真实小题诊断
-> 写入能力画像、作答证据、复习项
-> 回到今日任务
```

### 2.2 今日训练

今日任务必须由数据生成，不允许只展示静态推荐。输入包括：

- `StudyGoal`：考试日期、目标分、每日时间、优先技能。
- `SkillProfile`：入门诊断或训练产生的真实能力证据。
- `ReviewItem`：到期错因和词句复习项。
- 策略：高效模式或巩固模式。

任务优先级：

1. 没有能力证据时先做入门诊断。
2. 有到期复习项时先做复习。
3. 有薄弱技能时推荐对应训练。
4. 口语如果在目标中开启，保留短回合重说。
5. 词汇如果没有证据或分数低，必须能进入词汇听音练习。

### 2.3 基础内容库

GA 不能是空壳。首发必须至少内置：

- 阅读原创模拟材料。
- 听力训练材料。
- 写作任务。
- 翻译任务。
- CET-4 核心词汇材料，含释义、例句、语块和语音播报入口。

不内置未授权真题。题库扩展通过“材料导入”完成，导入材料必须校验结构。

### 2.4 专项练习

每个练习模块必须满足：

- 能从 UI 直接进入。
- 能完成至少一组真实作答。
- 有明确评分或反馈。
- 能写入 `PracticeSession`、`Attempt`、`SkillProfile`。
- 错误、低信心或低分项能进入 `ReviewItem`。

专项模块包括：

- 阅读：篇章、选择题、信心标记、错因反馈。
- 听力：播放、作答、信心标记、错因反馈。
- 写作：用户先产出答案，再获得结构化反馈。
- 翻译：用户先产出译文，再获得结构化反馈。
- 口语：任务、首次表达、反馈、二次重说、对比。
- 词汇：单词播报、例句播报、词义辨析、语块记忆、复习入队。

### 2.5 复习闭环

复习不应只是查看解析。真实复习必须包含：

```text
主动回忆
-> 挖空补全
-> 语境化输出
-> 更新掌握度
-> 更新下次复习时间
```

复习项来源包括：

- 阅读/听力错题。
- 写作/翻译反馈中的薄弱表达。
- 口语升级句。
- 词汇错误或低信心词。
- 入门诊断低分点。

### 2.6 能力地图

能力地图不能展示伪造分数。规则：

- 有 `SkillProfile` 才展示分数。
- 没有证据时显示“待诊断/待训练”。
- 分数必须来自真实诊断、练习、复习或设置保存。
- 用户完成训练后应能看到证据增长。

## 3. 页面要求

### 今日训练

- 不应看起来像营销或演示页。
- CTA 必须能进入真实功能。
- 无能力证据时强调诊断。
- 有复习项时强调复习。
- 有薄弱项时显示对应训练。

### 专项练习

- 顶部必须说明当前内置基础材料规模。
- “AI 英语能力教练”不能悬浮遮挡任何按钮。
- 词汇练习必须作为一等入口，不应只藏在复习里。
- 所有卡片按钮在 320px 手机宽度下可点击且无横向溢出。

### 入门诊断

- 不能只是自评。
- 必须有真实题目或任务。
- 必须能返回上一步。
- 不同答案必须产生不同能力分。
- 结果必须写入学习证据。

### 设置与数据

- 用户必须能导出本地学习数据。
- 用户必须知道清空浏览器数据会影响本地记录。
- 不需要邮箱交付、邮件验证或找回邮件。

## 4. 非目标

GA 阶段不承诺：

- 官方真题库。
- 大规模授权题库。
- 完整 CET-6/IELTS/TOEFL。
- 付费系统。
- 社区或排行榜。
- 邮件交付功能。

## 5. 当前阻塞定义

以下问题出现任意一项，不得认为达到 GA：

- 首次用户无法独立完成诊断。
- 今日任务不能进入真实训练。
- 练习完成后没有写入本地证据。
- 没有基础材料或词汇材料。
- 词汇练习没有语音播报。
- 复习队列无法执行主动回忆闭环。
- 移动端按钮遮挡、横向溢出或关键 CTA 不可点击。
- 产品文案暗示“官方真题库”或把离线兜底说成真实 AI。

## 6. 本轮补充验收要求

为避免再次把演示产品误判为可用产品，本轮验收以 `docs/ga-acceptance-checklist-2026-05-26.md` 为准，必须同时满足：

- 入门诊断不是自评：听力诊断必须有播报入口，用户可返回修改目标和答案。
- 今日任务不是静态清单：词汇证据为空或偏低时，必须自动安排词汇听音任务。
- 词汇练习不是附属功能：专项练习必须有一等入口，支持单词播报、例句播报、作答、评分和落库。
- 复习不是看解析：所有复习项都必须走主动回忆、挖空补全、语境输出和间隔更新。
- UI 不遮挡：能力诊断入口不能悬浮盖住后方按钮，移动端主体不能横向溢出。
